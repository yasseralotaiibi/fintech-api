Riyada Open Banking - Consent Demo

This static sandbox walks through the mock CIBA login and consent creation experience.

Steps

  1. Initiate a CIBA request via POST /api/ciba/auth/request.
  2. Approve the request with POST /api/mock/nafath/approve.
  3. Poll POST /api/ciba/auth/token until approved.
  4. Call GET /api/consents using the issued access token.
